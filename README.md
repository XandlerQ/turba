# Turba

A small experimental Godot prototype exploring the foundations of an agent-based swarm intelligence game.

## Current Features

- Basic agent entity setup
- Component-based agent architecture:
  - Sensing component
  - Behavior component
  - Locomotion component
  - Rotation component
- Locomotion state machine
- Behavior state machine
